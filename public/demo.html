Query 2: Get My Podcasts (Top 3 by Recent Content) 
query {
  podcasts(
    first: 3
    orderBy: "-latest_internal_episode_date"
  ) {
    edges {
      node {
        id
        uuid
        name
        latestInternalEpisodeDate
      }
    }
  }
}
Query 3: Get News for a Podcast

After running Query 2, this will auto-populate with your first podcast UUID

  news(podcastUuid: "YOUR_PODCAST_UUID") {
