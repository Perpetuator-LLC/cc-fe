# User Stories (or Standard Operating Procedures)

This directory contains a series of user stories that are intended to define the 'green path' for the project. These
stories are intended to be used as a reference for the development team to understand the expected behavior of the 
system. They are also intended to be used as a reference for the QA team to understand the expected behavior of the 
system. 

These have also been shared with users to direct them and help them understand the system. These documents 
should turn into proper user documentation for the system.
