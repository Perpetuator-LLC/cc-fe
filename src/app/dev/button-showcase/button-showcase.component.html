Debug Console Script

Run this in the browser console to inspect any button's computed styles:

// Select the Create Your Podcast Now button and inspect its styles
const btn = document.querySelector('button.btn-secondary');
const styles = getComputedStyle(btn);
console.log('Border Radius:', styles.borderRadius);
console.log('Background:', styles.backgroundColor);
console.log('All applied styles:', Object.fromEntries(
  ['borderRadius', 'backgroundColor', 'color', 'border', 'boxShadow']
    .map(p => [p, styles[p]])
));
 Expected Behavior
  • Border Radius: All buttons should have 8px border-radius (not pill-shaped)
  • .btn-primary: Blue background (#73adff dark / #005fb0 light)
  • .btn-secondary: Orange background (vibrant CTA)
  • .btn-tertiary: Purple background (AI features)
  • -outlined variants: Colored border, transparent/tinted background
  • -text variants: Just colored text, no background
  • -tonal variants: Container background color
