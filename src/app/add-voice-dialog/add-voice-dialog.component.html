Add Custom ElevenLabs Voice

Option 1: Using a PVC Shared Voice ID or 3rd Party Voice

If you have a Shared Voice ID, you can add it directly.
Note: If the voice owner changes any settings, this shared voice will also change.

Option 2: Complete Voice Import

For a more stable setup, provide both the voice ID and public user ID. This lets us import the complete voice, protecting it from future changes by the original owner.

Available Quality Models:

  • Premium HD (ElevenLabs Multilingual V2) - Highest quality
  • Premium (ElevenLabs Flash v2.5) - High quality

Tip: You can add the same voice multiple times with different quality models.
